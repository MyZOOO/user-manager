user-manager System
> version: 0.0.1

**这是一个典型的前后端分离的 Web 应用程序，后端使用 Java Spring Boot 框架，前端使用 Vue.js 框架。两者通过 API 进行数据交互。**

---

## **后端**

* **项目名称** **:** `backend`
* **项目描述** **:** `user-manager` **(用户管理)**
* **开发语言** **:** **Java**
* **编译器与运行环境** **: Java 8**

#### **核心框架**

* **Spring Boot** **: 作为整个项目的基石，版本为** `2.7.18`。它简化了配置和部署流程。
* **Spring Web** **: 通过** `spring-boot-starter-web` **提供，用于构建基于 RESTful API 的 Web 应用。**

#### **主要第三方库与功能**

* **安全框架** **:**
* **Apache Shiro** **: 版本** `1.10.0`，用于处理身份认证、授权等安全控制。
* **JWT (Json Web Token)** **: 使用** `jjwt` **库 (**`0.9.1`) 来实现无状态的用户会话管理。
* **数据持久层** **:**
* **MyBatis-Plus** **: 版本** `3.5.9`，一个强大的 MyBatis 增强工具，在 MyBatis 的基础上只做增强不做改变，简化了数据库操作和 CRUD 开发。
* **MySQL Driver** **: 使用** `mysql-connector-j` **(**`9.5.0`) 作为连接和操作 MySQL 数据库的驱动。
* **HikariCP** **: 作为默认的数据库连接池，提供高性能的数据库连接管理。*** **数据交换与工具库** **:**
* **FastJSON** **: 版本** `1.2.83`，阿里巴巴开源的 JSON 解析库，用于 Java 对象与 JSON 字符串之间的转换。
* **Lombok** **: 版本** `1.18.30`，通过注解（如 `@Data`, `@Getter`, `@Setter`）在编译期自动生成样板代码，减少冗余。
* **其他依赖** **:**
* **JAXB** **:** `jaxb-api` **和** `jaxb-runtime`，用于处理 XML，解决在新版本 Java 中移除该模块导致的兼容性问题。
* **Spring Boot Starter Test** **: 包含 JUnit、Mockito 等，用于单元测试。**

---

## **前端**

* **项目名称** **:** `frontend`
* **开发语言** **:** **JavaScript / TypeScript**
* **Node.js 版本要求** **:** `^20.19.0 || >=22.12.0` **(由** `engines.node` **指定，推荐使用符合要求的 LTS 版本)**

#### **核心框架**

* **Vue.js** **: 采用最新的 Vue 3 (**`^3.5.25`)，利用其 Composition API 进行组件开发，提供响应式的数据绑定和灵活的组件系统。
* **Vue Router** **: 版本** `^4.6.4`，官方路由管理器，用于构建单页面应用 (SPA)。

#### **主要 UI 库与功能**

* **Element Plus** **: 版本** `^2.12.0`，一套为开发者、设计师和产品经理准备的基于 Vue 3 的桌面端组件库，提供了丰富的预设界面组件，加速开发。
* **Icons** **:** `@element-plus/icons-vue` **(**`^2.3.2`)，为 Element Plus 提供的一套配套图标库。

#### **主要工具链与开发依赖**

* **构建工具** **:** **Vite** **(**`^7.3.0`)，一个现代化的前端构建工具，以其快速的冷启动和热更新而著称，显著提升开发体验。
* **HTTP 客户端** **:** **Axios** **(**`^1.13.2`)，一个基于 Promise 的 HTTP 客户端，用于向后端 API 发起请求。
* **开发辅助** **:**
* `@vitejs/plugin-vue`: Vite 官方提供的插件，用于支持 Vue 单文件组件 (`.vue` **文件)。**
* `vite-plugin-vue-devtools`: 为 Vite 项目集成 Vue DevTools，方便调试组件状态和性能。
